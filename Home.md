# Home

## Apache

Configuration files are located here:
```
/etc/apache2
```
Commands to operate the apache installation:

```bash
sudo apachectl start
sudo apachectl stop
sudo apachectl restart
```

```
   <editor>

      <custom_command>open -a jEdit $f</custom_command>
   </editor>
```
